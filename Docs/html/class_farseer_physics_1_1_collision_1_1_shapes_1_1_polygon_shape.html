Detailed Description

Represents a simple non-selfintersecting convex polygon.

Create a convex hull from the given array of points.

void FarseerPhysics.Collision.Shapes.PolygonShape.Set ( Vertices  input)

Copy vertices.

This assumes the vertices define a convex polygon. It is assumed that the exterior is the the right of each edge.

Warning
the points may be re-ordered, even if they form a convex polygon
collinear points are handled but not removed. Collinear points may lead to poor stacking behavior.
