Detailed Description

A fixture is used to attach a Shape to a body for collision detection.

A fixture inherits its transform from its parent. Fixtures hold additional non-geometric data such as friction, collision filters, etc. Fixtures are created via Body.CreateFixture. Warning: You cannot reuse fixtures.

short FarseerPhysics.Dynamics.Fixture.CollisionGroup
getset

Defaults to 0

If Settings.UseFPECollisionCategories is set to false: Collision groups allow a certain group of objects to never collide (negative) or always collide (positive). Zero means no collision group. Non-zero group filtering always wins against the mask bits.

If Settings.UseFPECollisionCategories is set to true: If 2 fixtures are in the same collision group, they will not collide.
