const float FarseerPhysics.Settings.AABBExtension = 0.1f

This is used to fatten AABBs in the dynamic tree.

This allows proxies to move by a small amount without triggering a tree adjustment. This is in meters.

const float FarseerPhysics.Settings.Baumgarte = 0.2f

This scale factor controls how fast overlap is resolved.

Ideally this would be 1 so that overlap is removed in one time step. However using values close to 1 often lead to overshoot.

bool FarseerPhysics.Settings.ConserveMemory = false
static

Conserve memory makes sure that objects are used by reference instead of cloned.

When you give a vertices collection to a PolygonShape, it will by default copy the vertices instead of using the original reference. This is to ensure that objects modified outside the engine does not affect the engine itself, however, this uses extra memory. This behavior can be turned off by setting ConserveMemory to true.
